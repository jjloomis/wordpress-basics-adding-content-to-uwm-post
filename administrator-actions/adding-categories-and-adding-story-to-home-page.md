# Adding Categories and Tags

1. In a post, make sure the **Post** tab is selected (on right.)
2. Scroll down to the **Categories** area (on right.)
3. Select the post's primary category (**News**, **Opinion**, **Sports**, etc.)
4. If you’d like the post to appear on the home page, select the **Home - Category Name** category (in addition to the post's primary category.)
5. If you’d like the post to appear at the top of a home page category area (area with featured image), make sure the post has a featured image (see [Setting a Featured Image](../working-with-media/adding-a-featured-image.md).) Then click the post's **Status** (**Draft**, **Private**, **Public**, etc.) In the **Status & visibility** fly-out menu, select **Sticky**.
6. When finished, click **Publish** or **Update**.
